Detailed Description

Skeleton is a passive container for a completely tracked skeleton. It does not do any tracking itself, instead relying on cv::Point objects passed in via the constructor. Skeleton objects are constructed by CommodityTracking itself from the getSkeleton method. Skeleton objects expose simple utility methods for accessing joint locations, but it also will optionally magnify these positions (to assist visualization). Magnification is controlled by setMagnification.
